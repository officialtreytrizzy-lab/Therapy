# Accessibility and auth delivery checklist

Before beta, verify VoiceOver/screen readers, keyboard navigation, visible focus, Dynamic Type/responsive text, reduced motion, contrast, form labels, landscape iPhone, iPad split-screen, older Safari, slow networks, low power mode, and Android Chrome. Passwordless email must have branded sender/template setup, resend, expiry copy, cross-device recovery, spam guidance, custom auth domain readiness, and support escalation.
