# incident response runbook

- Use correlation IDs to join browser/API/Firestore events.
- Review metadata-first audit events only; do not inspect private relationship text by default.
- Require admin role, least privilege, and a written reason for every support action.
- If safety, legal, or clinical questions arise, pause product claims and escalate to qualified external professionals.
- Roll back by reverting the feature flag or branch; do not run destructive migrations from local machines.
